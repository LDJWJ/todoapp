# Todo App

모던하고 프리미엄한 디자인의 할 일 관리 애플리케이션입니다...

https://ldjwj.github.io/todoapp/

## 🚀 기술 스택

- **HTML5** - 시맨틱 마크업
- **CSS3** - Glassmorphism 디자인, CSS Variables
- **Vanilla JavaScript** - 순수 자바스크립트 (프레임워크 없음)
- **LocalStorage** - 데이터 영구 저장
- **Font Awesome** - 아이콘
- **Google Fonts (Outfit)** - 타이포그래피

## 📋 주요 기능

### ✅ 기본 기능
- 할 일 추가/삭제
- 완료 상태 토글
- 남은 할 일 개수 표시
- 완료된 항목 일괄 삭제
- LocalStorage를 통한 데이터 영구 저장

### 🎨 고급 기능
- **다크/라이트 모드** - 테마 전환 및 설정 저장
- **인라인 편집** - 더블클릭으로 할 일 수정
- **필터링** - 전체/진행중/완료 상태별 필터
- **카테고리 시스템** - 색상별 카테고리 분류 및 필터링

## 🎯 사용 방법

### 기본 사용법
1. 입력창에 할 일을 입력하고 `+` 버튼을 클릭하거나 `Enter` 키를 누릅니다
2. 체크박스를 클릭하여 완료 상태를 토글합니다
3. 휴지통 아이콘을 클릭하여 할 일을 삭제합니다

### 고급 기능
- **테마 변경**: 우측 상단의 ☀️/🌙 아이콘 클릭
- **할 일 수정**: 할 일 텍스트를 더블클릭하여 편집 모드 진입
  - `Enter` - 저장
  - `Escape` - 취소
  - 포커스 해제 - 자동 저장
- **필터링**: 상단의 All/Active/Completed 탭 클릭
- **카테고리**: 할 일 추가 시 카테고리 선택 (업무/개인/쇼핑/건강/기타)
  - 카테고리별 색상 배지 자동 표시
  - 카테고리 필터 버튼으로 분류별 조회

- **Glassmorphism** - 반투명 유리 효과
